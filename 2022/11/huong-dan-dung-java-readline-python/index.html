Hàm float() trong python là gì

float() trong python là hàm dùng để ép kiểu giá trị được chỉ định thành dạng số thực dấu phẩy động trong python.

Chúng ta sử dụng hàm float() trong python để chuyển đổi giá trị được chỉ định thành số thực dấu phẩy động và tạo ra một số mới với cú pháp sau đây:

Cú pháp hàm float() trong Python

float(x)

Trong đó

  • float là tên hàm số
  • x là giá trị cần chuyển sang dạng float. x có thể là số hoặc là chuỗi chữ số.

Kết quả thu được sẽ là một số ở dạng số thực dấu phẩy động float.
